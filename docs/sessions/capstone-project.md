# 🏆 Capstone Project: The "Agentic" Hackathon

**Goal:** Synthesize learning by building a projects using Agentic workflows.

## Project Overview

The capstone project brings together all concepts from the three sessions into a comprehensive hands-on experience. You'll build a complete feature using agentic workflows, demonstrating mastery of the tools and techniques covered.

## Requirements

1. Use AI tool (Cursor/Windsurf), set up rules, skills, MCP server, workflows and what you learn so far — export chat logs
2. Write unit tests and Playwright tests using TDD
3. Keep project context in `context/` folder ( modules, functional)
4. Use git submodules for multi-repo
5. Write experience report — what worked, what didn't

## Submission Guidelines

1. **Code Repository**
   - Clean, well-documented codebase
2. **Documentation**
   - Complete `context/` folder
   - `EXPERIENCE.md` report
   - AI tool usage report
   - README with setup instructions

3. **Tests**
   - All tests passing
   - Coverage reports included
   - Test automation configured

4. **Chat Logs**
   - Exported AI chat logs
   - Demonstrates effective AI collaboration

## Evaluation Criteria

- **Technical Implementation (40%)**
  - Code quality and architecture
  - Test coverage and quality
  - Proper use of git submodules

- **AI Tool Mastery (20%)**
  - Effective configuration and usage
  - Quality of chat logs
  - Advanced feature utilization

- **Documentation (20%)**
  - Completeness of context folder
  - Clarity of experience report
  - Quality of technical documentation

- **Reflection & Learning (20%)**
  - Depth of insights in experience report
  - Honest assessment of challenges
  - Constructive feedback and recommendations

Good luck with your capstone project! 🚀
